## Hot keys

- ctrl + shift + ` = New terminal
- cmd + n = new file
- cmd + z = undo
- cmd + shift + z = redo
- alt + shift + up = copy line up
- alt + up = move line up
- alt + cmd + up = add cursor below
- cmd + p = go to file
- cmd + f = find in file
- F5 = start debuging
- cmd + q = close program
